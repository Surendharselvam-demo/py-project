The PDF to Audiobook Converter is a simple web-based application that allows users to upload a PDF document and convert its text content into an MP3 audio file. This project focuses on accessibility and ease of use, helping users listen to documents instead of reading them.

The application provides a clean and user-friendly interface built using HTML and CSS and is designed to integrate easily with a backend that handles PDF processing and text-to-speech conversion.
